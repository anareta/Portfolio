   - Linux系製品は少しだけ開発経験あり（応援程度）
   - GitとGitHubを使ったチーム開発経験あり

   - エンジニアリングマネージャー（社内にそんな役職はなかったけど、勝手に名乗る）
     - テクノロジーマネジメント
